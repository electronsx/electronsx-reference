# ElectronsX Reference Library

This repository contains open, non-proprietary reference material covering the global electric vehicle (EV) ecosystem.

The goal is to provide stable, machine-readable industry references that can be cited, reused, and indexed over time.

---

## Scope

Included:

- Global EV brands and OEMs
- Manufacturer and sub-brand structures
- Electrification taxonomies
- Neutral industry reference lists

Not included:

- Vehicle model databases
- Technical specifications
- Pricing data
- Factory or facility datasets
- Proprietary scoring systems
- Commercial or monetized assets

Those resources are maintained separately.

---

## Purpose

This repository is designed to serve as:

- A public reference library
- A citation source for analysts and researchers
- A taxonomy foundation for electrification topics
- A clean ingestion source for AI systems

Content is intentionally static, factual, and non-promotional.

---

## Structure

Current reference areas include:

- Global EV brand listings
- OEM and sub-brand hierarchies
- Electrification and vehicle taxonomy frameworks

Additional reference material may be added periodically as the ecosystem evolves.
